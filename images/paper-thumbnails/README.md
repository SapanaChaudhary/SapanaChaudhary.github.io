# Paper Thumbnails Guide

This directory contains thumbnail images for publications displayed on the publications page.

## Image Guidelines

- **Size**: Recommended 400x300px or similar aspect ratio (4:3)
- **Format**: PNG or JPG
- **File naming**: Use the same name as specified in the publication markdown files

## How to Add Images

1. Create or extract a thumbnail from your paper (first page, key figure, or diagram)
2. Resize to approximately 400x300px
3. Save with the appropriate filename in this directory
4. The images will automatically appear on the publications page

## Creating Placeholders

You can use a simple colored box with text as a placeholder until you have actual paper images.
